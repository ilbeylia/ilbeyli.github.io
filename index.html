Ilbeyli v2.0 Control Card

The Ilbeyli v2.0 control card is a powerful microcontroller-based control board designed for integration with a wide range of motors and sensors. This board is intended for various industrial and hobby applications such as CNC machines, laser cutting machines, 3D printers, robots, and other automation projects. The Ilbeyli v2.0 control card features the following:

  • STM32F411 Microcontroller: Equipped with a high-performance ARM Cortex-M4 processor, the Ilbeyli v2.0 offers fast data processing capabilities and a large memory capacity.
  • Step Motor Drivers: Provided to control high-torque stepper motors compatible with drivers such as TB6600, DRV8825, A4988, enabling precise motion control for applications like CNC machines and 3D printers.
  • Servo Motor Connections: Connection points are available for servo motors, allowing for precise and accurate position control suitable for robotics applications and manipulators.
  • Linear Motor Connections: The Ilbeyli v2.0 provides necessary connection points for controlling linear motors, essential for applications requiring fast and accurate linear movements.
  • Nextion Screen Support: Compatible with Nextion touchscreens, facilitating interactive user interface development for easy project control and monitoring.
  • Wide Range of Connection Options: The Ilbeyli v2.0 control card offers a wide range of pin options, allowing for easy integration of sensors, additional motors, or other components.

Applications:

  • CNC machines
  • 3D printers
  • Laser cutting machines
  • Robotic arms and manipulators
  • Automation systems
  • Process control systems

Benefits:

  • High-performance microcontroller
  • Wide range of connection options
  • Precise motion control
  • Easy integration and programming
  • Suitable for various industrial and hobby applications

The Ilbeyli v2.0 control card provides a robust platform offering flexibility and reliability for a wide range of applications. It serves as an ideal solution for CNC machines, laser cutting machines, robots, and other automation projects.

KeyStrVis - Keyboard Stroke Visualizer

KeyStrVis is a keyboard stroke visualizer that provides a dynamic and engaging representation of your typing activity. Whether you are a content creator, programmer, or just curious about your typing patterns, KeyStrVis offers a visually appealing way to explore and analyze your keyboard input.

Features

  • Real-time visualization of key presses and releases.
  • Customizable color schemes.
  • Support for various keyboard layouts.
  • Adjustable playback speed for replaying typing sessions.
  • Export and share your visualized typing sessions.

This project is being developed to the extent my spare time allows. The project is in progress, but color changes and themes are being developed.

Control And Data Manager

The main purpose of Control and Data Manager (CDM) is to facilitate data exchange with devices that need to be controlled in industrial facilities, factories, and similar environments, and to effectively manage this data. While the main module connects to the network via Ethernet, providing extensive control capabilities, the Control and Data Modules can communicate with the main module via WiFi to exchange data with various devices. Additionally, the Control and Data modules can perform data exchange without using the main module through a user interface over WiFi, allowing for flexible usage depending on the requirements. With Control and Data Manager (CDM), it becomes possible to monitor, control, and optimize production processes.

Industrial applications of Control and Data Manager (CDM) may include:

  • Automation and monitoring of production lines.
  • Collection and processing of sensor data.
  • Remote monitoring and control of machinery and equipment.
  • Energy management and optimization.
  • Industrial Internet of Things (IoT) applications.
  • Data collection and analysis systems.
  • Security systems and early warning systems.
